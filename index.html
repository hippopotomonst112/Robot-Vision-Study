// 01. about

프로젝트 소개

NANA는 Naturalized Advanced Network Assistant의 약자로, 자연스러운 대화형 로봇 제어 시스템입니다.

사용자의 텍스트 입력을 분석하여 로봇 하드웨어 제어, 센서 정보 안내, 로봇공학 지식 응답 등 다양한 기능을 수행합니다.

Flask 백엔드와 순수 HTML/CSS/JS 프론트엔드로 구성되어 있으며, Render 플랫폼을 통해 24시간 서비스됩니다.

TECH STACK

Python 3 Flask JavaScript HTML5 CSS3 REST API Render GitHub
